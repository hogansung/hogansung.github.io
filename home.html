This is a completely static website, which is based on bootstrap pre-built elements. There are five main webpages, includes

  • home
  • blog
  • undergrad project
  • graduate project
  • research
  • about

The wepages for index , home , and about are easy. However, I want to elaborate my works on blog and research .

How to build Blog page?

The core idea for blog is that it should at least support font-style control, and image attachment. Furthermore, there should be a main entrance for all blog articles. To achieve this requirement automatically, I decided to provide a friendly API that it takes markdown as input and generates one complete html page for article and partial html codes for preview, respectively as output.

How to build Research page?

The main concept for research page is to present my previous works in student projects and studies on specific topics. Thus, to illustrate my work in detail, I should provide at least brief description , simple figure , and source code (if possible). To fulfill the requirement with less human work, I determined to create another friendly API that it will take readme , image, pdf, and zip files (if possible) as input and generates a complete html page as output.

How long did it take for me to build this site?

Actually, it took me less than 10 hours to finish the wirtten codes. However, I spent more than 15 hours designing and examining possible frameworks for my website. For example, I tried SemanticUI and Json but in vain. In my following works, most of the time was spent on being familiar with bootstrap framework.
